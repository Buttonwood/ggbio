scale_x_sequnit {ggbio}R Documentation

scale x by unit

Description

scale x by unit 'Mb','kb', 'bp'.

Usage

scale_x_sequnit(unit = c("Mb", "kb", "bp"))

Arguments

unit

unit to scale x.

Value

'position_c'

Author(s)

Tengfei Yin

Examples

library(ggplot2)
p <- qplot(x = seq(1, to = 10000, length.out = 40), y = rnorm(40), geom
= "point")
## default mb
p + scale_x_sequnit()


p + scale_x_sequnit("kb")


p + scale_x_sequnit("bp")
